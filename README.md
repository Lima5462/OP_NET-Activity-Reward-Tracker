OP_NET Activity & Reward Tracker

A beginner-friendly dashboard built for OP_NET Testnet users on Bitcoin Layer 1.

Overview

This app tracks wallet activity, estimates reward points, and analyzes airdrop eligibility for OP_NET users.

Features

- Single wallet tracking
- Daily transaction monitoring
- Points estimation
- Airdrop eligibility checker
- Simple dashboard UI

Network

OP_NET Testnet (Bitcoin Layer 1)

Vibecoding Project

This project is created using the vibecoding approach where the idea is described and the app structure is generated with AI assistance.
