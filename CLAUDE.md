Project: OP_NET Activity & Reward Tracker

Description

This is a beginner-friendly dashboard for OP_NET Testnet users on Bitcoin Layer 1.
It tracks wallet activity, estimates reward points, and analyzes airdrop eligibility safely.

Core Features

- Single wallet activity tracking
- Daily transaction analytics
- Points estimation dashboard
- Airdrop eligibility checker
- Clean and simple UI

User Focus

Designed for single wallet users to avoid multi-wallet risks and maintain safe engagement.

Security Design

- No private key storage
- Read-only analytics
- No automation or bot usage
- Safe interaction with OP_NET smart contracts

Target Network

OP_NET Testnet (Bitcoin Layer 1)
